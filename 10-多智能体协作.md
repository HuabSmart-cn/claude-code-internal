# 多智能体协作

## 1. AgentTool 概述

### 1.1 核心文件

主要文件位于 `/src/tools/AgentTool/`

### 1.2 Agent 定义结构

```typescript
export type AgentDefinition = {
  agentType: string
  whenToUse?: string
  disallowedTools?: string[]
  tools?: string[] | '*'
  model?: string
  permissionMode?: PermissionMode
  maxTurns?: number
  skills?: string[]
  mcpServers?: MCPServerSpec[]
}
```

## 2. 子智能体生成

### 2.1 runAgent 函数

创建和运行子智能体的核心函数，返回 AsyncGenerator。

### 2.2 上下文隔离

为子智能体创建隔离的上下文，包括独立的文件状态缓存和中止控制器。

## 3. 智能体生命周期管理

### 3.1 生命周期阶段

1. 初始化阶段 - 创建 Agent ID、设置上下文
2. 执行阶段 - 运行查询循环、处理工具调用
3. 清理阶段 - 清理 MCP 服务器、释放内存

## 4. 内置智能体 (built-in/)

### 4.1 generalPurposeAgent

通用目的 Agent，适用于复杂研究和多步骤任务执行。

### 4.2 planAgent

软件架构师 Agent，用于设计实现计划。**只读模式**：禁止任何文件修改操作。

### 4.3 verificationAgent

验证 Agent，用于验证任务完成情况。

### 4.4 exploreAgent

探索 Agent，用于代码库探索和搜索。**只读操作**。

## 5. 团队协调器 (coordinator/)

协调器模式允许多个 Agent 在团队中协作。

## 6. 团队管理工具

- TeamCreateTool - 创建团队
- TeamDeleteTool - 删除团队
