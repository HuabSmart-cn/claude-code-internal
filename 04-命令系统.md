# Claude Code 命令系统详解

## 概述

Claude Code 的命令系统是用户与 CLI 交互的核心方式。每个斜杠命令都是独立的模块，支持懒加载。

## 命令类型

### 1. prompt 类型命令
生成提示词发送给 AI 模型处理。

### 2. local-jsx 类型命令
渲染本地 React 组件的交互式命令。

### 3. local 类型命令
执行本地逻辑的命令，不涉及 AI 模型。

## 命令详解

### /commit - 创建 Git 提交
分析当前 Git 状态，生成符合项目规范的提交信息。

### /review - 代码审查
审查 GitHub Pull Request 的代码变更。

### /compact - 对话压缩
清除对话历史但保留摘要，保持在上下文中。

### /mcp - MCP 服务器管理
管理 MCP 服务器连接。

### /config - 配置面板
打开配置面板，管理 Claude Code 设置。

### /doctor - 诊断检查
诊断和验证 Claude Code 安装状态。

### /login / logout - 登录/登出
使用 Anthropic 账户登录或切换账户。

### /memory - 记忆文件编辑
编辑 Claude 记忆文件。

### /skills - 技能列表
列出可用的技能。

### /tasks - 后台任务管理
列出和管理后台任务。

### /vim - Vim 模式切换
在 Vim 和 Normal 编辑模式之间切换。

### /diff - 变更查看
查看未提交的变更和每轮对话的差异。

### /cost - 成本显示
显示当前会话的总成本和持续时间。

### /theme - 主题切换
更改终端主题。

### /context - 上下文可视化
将当前上下文使用情况可视化为彩色网格。

### /pr_comments - PR 评论获取
获取 GitHub Pull Request 的评论。

### /resume - 恢复对话
恢复之前的对话。

### /share - 分享会话
分享当前会话。

### /desktop - 桌面客户端
在 Claude Desktop 中继续当前会话。

### /mobile - 移动应用
显示下载 Claude 移动应用的二维码。

## 命令注册机制

命令通过 `src/commands.js` 统一注册和管理。

## 懒加载机制

大多数命令使用懒加载模式，确保只有实际使用的命令才会加载其代码。

## 权限控制

某些命令支持 `isEnabled()` 方法动态控制可用性。
