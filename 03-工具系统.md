# 工具系统详解

本文件说明：详细介绍 Claude Code 的工具系统，包括所有内置工具的功能、输入 schema、权限模型和核心逻辑。

## 1. 工具系统架构

### 1.1 工具注册表

所有内置工具在 `tools.ts` 中注册，通过 `getAllBaseTools()` 获取。

### 1.2 工具调用流程

```
用户输入/模型决策 → checkPermissions() → validateInput() → call() → renderToolResultMessage()
```

## 2. BashTool - Shell 命令执行

### 2.1 功能描述

BashTool 是 Claude Code 最核心的工具之一，负责在用户授权下执行 Shell 命令。

### 2.2 权限模型

- 检查命令是否匹配 alwaysAllowRules
- 检查命令是否匹配 alwaysDenyRules
- 检查命令是否属于危险命令
- 用户确认后执行

### 2.3 沙箱执行

Claude Code 支持使用沙箱执行危险命令。

## 3. FileReadTool - 文件读取

### 3.1 功能描述

FileReadTool 用于读取文件内容，支持多种文件类型：纯文本、图片、PDF、Jupyter Notebooks。

## 4. FileWriteTool - 文件写入

### 4.1 功能描述

FileWriteTool 用于创建或覆盖文件内容。

## 5. FileEditTool - 文件编辑

### 5.1 功能描述

FileEditTool 支持两种编辑模式：Search/Replace 模式和直接编辑模式。

## 6. GlobTool - 文件模式匹配

使用 glob 模式查找文件。

## 7. GrepTool - 代码搜索

在代码库中搜索匹配文本，支持正则表达式。

## 8. WebFetchTool - 网页获取

获取网页内容，支持预批准域名列表。

## 9. WebSearchTool - 网络搜索

执行网络搜索。

## 10. AgentTool - 子 Agent

AgentTool 允许 Claude 创建和管理子 Agent 来并行执行任务。

## 11. SkillTool - 技能执行

SkillTool 用于执行用户定义的技能。

## 12. MCPTool - Model Context Protocol

MCPTool 是 MCP 协议的工具包装器。

## 13. 工具权限系统总结

### 13.1 权限模式

| 模式 | 描述 | 行为 |
|------|------|------|
| default | 默认模式 | 询问危险操作 |
| auto | 自动模式 | 基于规则自动决定 |
| bypass | 绕过模式 | 允许所有操作 |
| plan | 计划模式 | 强制计划模式 |

### 13.2 权限检查流程

1. validateInput() - 验证输入有效性
2. checkPermissions() - 检查权限规则
3. 需要询问 → 等待用户确认

## 14. 总结

Claude Code 的工具系统设计遵循：一致性、安全性、可扩展性、用户控制、高性能。
