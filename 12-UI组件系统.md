# UI 组件系统

## 目录结构

```
src/
├── components/           # React 业务组件 (~140个)
├── ink/                 # Ink 渲染器封装
│   ├── ink.tsx         # 主渲染器
│   ├── components/     # Ink 基础 UI 组件
│   ├── events/         # 事件系统
│   ├── hooks/          # React Hooks
│   └── layout/         # Yoga 布局引擎封装
├── screens/            # 全屏 UI 屏幕
└── outputStyles/       # 输出样式管理
```

## 1. Ink 渲染器 (ink/)

Ink 是一个用于构建终端 UI 的 React 渲染器。

### 1.1 主渲染器 (ink.tsx)

核心类 Ink 的主要职责：
- 管理 React Fiber 树和渲染生命周期
- 处理终端输入/输出
- 管理帧缓冲区和屏幕更新

### 1.2 Ink 组件

- **Box** - 核心布局组件，类似 Flexbox
- **Button** - 交互式按钮组件
- **ScrollBox** - 支持滚动的容器组件
- **Text** - 文本显示
- **Link** - 超链接渲染

## 2. React 业务组件 (components/)

components 目录包含约 140 个 React 组件，提供高级 UI 功能。

### 2.1 主要分类

| 类别 | 示例组件 | 用途 |
|------|----------|------|
| 消息 | Message.tsx | 消息显示 |
| 对话 | ChatInput.tsx | 用户输入 |
| 设置 | Settings/* | 设置界面 |
| 诊断 | Doctor.tsx | 问题诊断 |
| 权限 | permissions/* | 权限管理 |
| MCP | mcp/* | MCP 服务器 |
| 任务 | TaskListV2.tsx | 任务管理 |

## 3. 全屏屏幕 (screens/)

- **Doctor.tsx** - 诊断屏幕
- **REPL.tsx** - REPL 交互界面
- **ResumeConversation.tsx** - 对话恢复屏幕

## 4. 核心概念

### 4.1 帧缓冲区

每帧包含完整的屏幕内容，由 Ink 渲染器管理双缓冲。

### 4.2 虚拟滚动

ScrollBox 通过只渲染可见内容实现虚拟滚动。

### 4.3 Yoga 布局

使用 Yoga 引擎计算 Flexbox 布局。
