# suggestions/ - 建议系统

## 功能概述

`suggestions/` 目录提供各种建议和自动补全功能。

## 文件列表

```
suggestions/
├── commandSuggestions.ts    # 命令建议
├── directoryCompletion.ts   # 目录补全
├── shellHistoryCompletion.ts  # Shell 历史补全
├── skillUsageTracking.ts    # 技能使用跟踪
└── slackChannelSuggestions.ts  # Slack 频道建议
```

## 建议类型

### 命令建议
根据上下文建议可能的命令。

### 目录补全
目录路径自动补全。

### Shell 历史
基于 Shell 历史的建议。
