# CustomSelect/

## 功能
自定义选择器组件库，提供高级选择功能。

## 组件列表

- `select.tsx` - 主选择器组件（115KB）
- `SelectMulti.tsx` - 多选组件（29KB）
- `select-input-option.tsx` - 输入选项组件
- `select-option.tsx` - 选项组件
- `use-select-state.ts` - 选择状态 Hook
- `use-multi-select-state.ts` - 多选状态 Hook
- `use-select-navigation.ts` - 选择导航 Hook
- `use-select-input.ts` - 选择输入 Hook
- `option-map.ts` - 选项映射
- `index.ts` - 导出

## 重要组件

### select
通用的下拉选择器组件，支持：
- 模糊搜索
- 键盘导航
- 自定义渲染
- 分组选项

### SelectMulti
多选版本的 select 组件。
