# hooks/

## 功能
React Hooks 组件库，提供各种 UI 相关的 Hook。

## 组件列表

- `HooksConfigMenu.tsx` - Hook 配置菜单（54KB）
- `PromptDialog.tsx` - 提示对话框
- `SelectEventMode.tsx` - 选择事件模式
- `SelectHookMode.tsx` - 选择 Hook 模式
- `SelectMatcherMode.tsx` - 选择匹配器模式
- `ViewHookMode.tsx` - 查看 Hook 模式

## 重要组件

### HooksConfigMenu
Hook 配置界面：
- 查看已配置的 Hook
- 添加新 Hook
- 配置 Hook 行为

## Hooks 架构说明

这些组件用于管理 Claude Code 的 Hook 系统，允许用户配置在特定事件发生时自动执行的脚本。
