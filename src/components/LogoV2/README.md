# LogoV2/

## 功能
Logo 和品牌组件库。

## 组件列表

- `LogoV2.tsx` - 主 Logo 组件（75KB）
- `Clawd.tsx` - Clawd Logo
- `AnimatedClawd.tsx` - 动画 Clawd
- `CondensedLogo.tsx` - 紧凑 Logo
- `WelcomeV2.tsx` - 欢迎界面（57KB）
- `Feed.tsx` - Feed 组件
- `FeedColumn.tsx` - Feed 列
- `feedConfigs.tsx` - Feed 配置
- `ChannelsNotice.tsx` - 频道通知
- `GuestPassesUpsell.tsx` - 访客通行证促销
- `OverageCreditUpsell.tsx` - 超额信用促销
- `VoiceModeNotice.tsx` - 语音模式通知
- `EmergencyTip.tsx` - 紧急提示
- `Opus1mMergeNotice.tsx` - Opus 1M 合并通知
- `AnimatedAsterisk.tsx` - 动画星号

## 重要组件

### LogoV2
主 Logo 组件，包含品牌的所有视觉变体。

### WelcomeV2
欢迎界面，在应用启动时显示。
