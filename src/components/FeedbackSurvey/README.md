# FeedbackSurvey/

## 功能
反馈调查组件，收集用户对使用体验的反馈。

## 组件列表

- `FeedbackSurvey.tsx` - 反馈调查主组件
- `FeedbackSurveyView.tsx` - 调查视图
- `TranscriptSharePrompt.tsx` - 转录分享提示
- `useFeedbackSurvey.tsx` - 调查 Hook（48KB）
- `useMemorySurvey.tsx` - 记忆调查 Hook（30KB）
- `usePostCompactSurvey.tsx` - Compact 后调查 Hook
- `useSurveyState.tsx` - 调查状态 Hook
- `useDebouncedDigitInput.ts` - 数字输入 Hook
- `submitTranscriptShare.ts` - 转录提交

## 重要组件

### FeedbackSurvey
反馈调查对话框，引导用户：
- 评价使用体验
- 描述问题或建议
- 选择是否愿意分享对话记录
