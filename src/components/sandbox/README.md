# sandbox/

## 功能
Sandbox 配置和管理 UI。

## 组件列表

- `SandboxSettings.tsx` - Sandbox 设置（29KB）
- `SandboxConfigTab.tsx` - 配置标签页
- `SandboxDependenciesTab.tsx` - 依赖标签页
- `SandboxOverridesTab.tsx` - 覆盖标签页
- `SandboxDoctorSection.tsx` - 诊断区块

## 重要组件

### SandboxSettings
Sandbox 环境配置界面：
- 配置 Sandbox 执行环境
- 管理依赖
- 设置覆盖规则
- 诊断问题

### SandboxDoctorSection
Sandbox 诊断工具，帮助用户排查配置问题。
