# Spinner/

## 功能
加载动画组件库。

## 组件列表

- `SpinnerAnimationRow.tsx` - 动画行
- `SpinnerGlyph.tsx` - 旋转符号
- `TeammateSpinnerLine.tsx` - 队友旋转行
- `TeammateSpinnerTree.tsx` - 队友旋转树
- `GlimmerMessage.tsx` - 微光消息
- `FlashingChar.tsx` - 闪烁字符
- `ShimmerChar.tsx` - 微光字符
- `useStalledAnimation.ts` - 停滞动画 Hook
- `useShimmerAnimation.ts` - 微光动画 Hook
- `utils.ts` - 工具函数
- `index.ts` - 导出
- `teammateSelectHint.ts` - 队友选择提示

## 重要组件

### TeammateSpinnerTree
树形旋转动画，用于展示多层级任务进度。

### GlimmerMessage
微光效果的消息动画。
