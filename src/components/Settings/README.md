# Settings/

## 功能
应用设置界面，提供配置管理、状态展示和使用统计。

## 组件列表

- `Config.tsx` - 配置面板（271KB，超大组件）
- `Settings.tsx` - 设置主界面
- `Status.tsx` - 状态面板
- `Usage.tsx` - 使用统计

## 重要组件

### Config
核心配置组件，处理所有应用设置的展示和编辑，包括：
- 模型选择
- 主题配置
- 快捷键设置
- 代理设置
- 安全设置

### Usage
使用统计界面，展示：
- API 使用量
- Token 消耗
- 成本统计

### Status
状态面板，显示：
- 连接状态
- 当前 Agent
- 活跃任务
