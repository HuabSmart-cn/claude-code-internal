# AgentTool/built-in/ - 内置代理

## 功能

预定义的代理实现。

## 核心文件

| 文件 | 功能 |
|------|------|
| `generalPurposeAgent.ts` | 通用目的代理，处理日常任务 |
| `planAgent.ts` | 计划模式代理，用于复杂任务设计 |
| `exploreAgent.ts` | 探索代理，用于代码库探索 |
| `verificationAgent.ts` | 验证代理，用于验证实现正确性 |
| `claudeCodeGuideAgent.ts` | Claude Code 使用指南代理 |
| `statuslineSetup.ts` | 状态栏设置 |

## generalPurposeAgent

默认代理类型，处理各类任务。

## planAgent

用于 Plan Mode，提供任务分析和计划。

## exploreAgent

用于快速了解代码库结构。

## verificationAgent

用于验证代码变更的正确性。
