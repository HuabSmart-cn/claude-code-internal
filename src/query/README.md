# query

## 功能
查询配置模块，负责管理查询相关的配置和生命周期。

## 文件说明

- `config.ts` - 查询配置
- `deps.ts` - 依赖定义
- `stopHooks.ts` - 停止钩子 (17KB)
- `tokenBudget.ts` - Token 预算管理

## 核心概念

### 查询配置
管理 AI 查询的参数和行为设置。

### Token 预算
- 单次查询的 token 上限
- 对话总 token 预算
- 超出预算时的处理策略

### 停止钩子
定义查询停止时的回调和处理逻辑：
- 资源清理
- 状态保存
- 错误处理

### 依赖管理
模块间的依赖关系和初始化顺序。
