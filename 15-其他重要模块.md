# 其他重要模块

## 目录结构

```
src/
├── bootstrap/        # 启动引导
├── memdir/          # 持久化内存目录
├── native-ts/       # 原生 TypeScript 工具
├── voice/           # 语音输入
├── vim/             # Vim 模式
├── keybindings/     # 快捷键配置
├── moreright/       # 权限扩展 UI
├── outputStyles/    # 输出样式管理
└── query/           # 查询管道
```

## 1. 启动引导 (bootstrap/)

管理应用启动状态和全局状态，包括启动时间跟踪、交互时间统计、功能开关状态、工作目录管理。

## 2. 持久化内存 (memdir/)

### 2.1 内存目录核心

管理 MEMORY.md 文件和自动记忆系统。

### 2.2 关键常量

```typescript
export const ENTRYPOINT_NAME = 'MEMORY.md'
export const MAX_ENTRYPOINT_LINES = 200
export const MAX_ENTRYPOINT_BYTES = 25_000
```

## 3. 原生 TypeScript 工具 (native-ts/)

### 3.1 Yoga 布局引擎

封装 Facebook Yoga 布局引擎。

### 3.2 颜色差异

计算颜色之间的视觉差异 (Delta E)。

## 4. 语音输入 (voice/)

检查语音输入模式是否启用。

## 5. Vim 模式 (vim/)

### 模块结构

- motions.ts - 动作 (h/j/k/l/w/b/e)
- operators.ts - 操作符 (d/y/c/s/r)
- textObjects.ts - 文本对象
- transitions.ts - 状态转换

### 状态机

- normal - 普通模式
- insert - 插入模式
- visual - 可视模式
- operator - 操作符等待

## 6. 快捷键配置 (keybindings/)

- defaultBindings.ts - 默认绑定
- parser.ts - 解析器
- validate.ts - 验证
- useKeybinding.ts - Hook

## 7. 查询管道 (query/)

### 7.1 Token 预算管理

管理查询的 Token 预算。

### 7.2 停止钩子

定义查询停止时的回调钩子。

## 8. 总结

这些模块提供了 Claude Code 的重要辅助功能：启动引导、持久化记忆、原生工具集成、语音输入、Vim 模式、快捷键系统等。
